== Description ==
Display the page template name which is assigned to the page type on the page listing, also add an admin menu for showing all page template listings that we have added.

== Installation ==

Installation steps

= The Good Way =

1. In your WordPress Admin, go to the Add New Plugins page
1. Search for: Show All Template
1. Show Current Template should be the first result. Click the Install link.

= The Old Way =

1. Upload the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where can I find listings of page templates? =
You can see the all-page template listing in settings->page template.
